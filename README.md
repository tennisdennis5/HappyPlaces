# Happy Places App

## Beschreibung

Mit der App können Nutzer ihre Lieblingsorte speichern und verwalten.

## Funktionen

- Orte mit Name und Beschreibung speichern
- Foto zu einem Ort hinzufügen (falls vorhanden)
- Standort auswählen
- Alle gespeicherten Orte werden auf einer Karte angezeigt
- Persönliche Notizen zu Orten können auch nach Belieben hinzugefügt oder entferntc werden

## Verwendete Technologien

- Kotlin
- Jetpack Compose
- Room Database
- OpenStreetMap (osmdroid)
- Android Studio
